Introduction

In this tutorial, we will cover the fundamentals of performing an NCA analysis with PuMaS of an example dataset in which a single intravenous (IV) bolus dose was administered.

The dataset

A single IV bolus dose of 2000 mg was administered to 24 different subjects. Samples were collected every 30 minutes until 21 hours after dose administration. Two different datasets are available, one with 7% of the samples below the limit of quantification (BLQ) and the other with 18% of the samples BLQ.

Let's start reading the dataset. By using the missingstring option we are specifying how the missing values are labeled in our dataset.

data7BLQ = CSV.read("./data/single_dose_IVbolus_7BLQ.csv",missingstring="NA")

Defining the units

Next we can define time, concentration and dose units so the report includes the units for the pharmacokinetic parameters.

timeu = u"hr"
concu = u"mg/L"
amtu  = u"mg"

mg

Defining the population object

Using the parse_ncadata function, the next step would be to define the population that we are going to use for the NCA. Within this function we need to specify the dataset, the name of the column with the subject identifier (id=), name of the time column (time=), name of the concentration column (conc=), name of the dose column (amt=), the interdose interval (ii=) multiplied by the time units, the name of the formulation column (formulation=) and the route of administration (route=).

pop7 = parse_ncadata(data7BLQ, id=:ID, time=:time, conc=:DV, amt=:DOSE, ii=24timeu,
    formulation=:Formulation, route=(iv="IV",), timeu=timeu, concu=concu, amtu=amtu)
pop18 = parse_ncadata(data18BLQ, id=:ID, time=:time, conc=:DV, amt=:DOSE, ii=24timeu,
    formulation=:Formulation, route=(iv="IV",), timeu=timeu, concu=concu, amtu=amtu)

Please, note that in the route=(iv="IV",) option the name between quotes should match the contents of the formulation column in our dataset.

Also note that in the function above by default the lower limit of quantification (LLQ) is 0 and concentrations that are below LLQ (BLQ) are dropped.

Let's say we want to specify an LLQ value of 0.4 mg/L, then we need to add llq=0.4concu to the function above:

pop7 = parse_ncadata(data7BLQ, id=:ID, time=:time, conc=:DV, amt=:DOSE, ii=24timeu,
    formulation=:Formulation, route=(iv="IV",), timeu=timeu, concu=concu, amtu=amtu,llq=0.4concu)
